## 🔑 关键问题解答

### Q1: 环境变量注入解密密钥是什么意思？

**答**: 
```
1. 首次运行技能时，自动生成密钥
2. 密钥保存在用户本地：~/.virtual_user/.key
3. 这个文件不会提交到 GitHub
4. 场景库数据用这个密钥加密
5. 运行时自动用本地密钥解密

好处：
- 加密数据可以安全提交到 GitHub
- 普通用户拿到代码，没有密钥，打不开数据
- 部署技能的人，首次运行自动生成密钥
```

## 📊 技术亮点

### 1. 向量化检索

- 模型：bge-large-zh-v1.5（中文优化）
- 算法：余弦相似度
- 性能：1000 条数据检索<1 秒

### 2. 加密保护

- 算法：Fernet（对称加密）
- 密钥：44 字节，随机生成
- 安全：文件权限 600（仅所有者可读写）

### 3. 模块化设计

```
main.py (主入口)
  ├─ encrypt.py (加密)
  ├─ vector_search.py (检索)
  ├─ user_generator.py (生成)
  └─ conversation.py (对话)
```
