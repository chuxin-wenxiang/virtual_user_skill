## 🎯 核心功能

### 功能流程

```
用户提问 → 检索场景库 → 生成 10 个虚拟用户类型 → 用户选择 → 虚拟用户对话 → 生成报告
```

### 三大阶段

| 阶段 | 功能 | 输出 |
|:---|:---|:---|
| **阶段 1** | 检索场景库，生成虚拟用户类型 | 10 个用户类型列表 |
| **阶段 2** | 用户选择/反选目标人群 | 选中的用户类型 |
| **阶段 3** | 虚拟用户对话 | 对话记录 + 报告 |

## 📊 技术架构

### 模块依赖

```
main.py (主入口)
  ├─ encrypt.py (加密管理)
  ├─ vector_search.py (向量检索)
  │   └─ sentence-transformers (向量化模型)
  ├─ user_generator.py (用户生成)
  └─ conversation.py (对话管理)
```

### 数据流

```
用户问题
  ↓
向量化 (bge-large-zh-v1.5)
  ↓
相似度搜索 (cosine_similarity)
  ↓
场景匹配 (top 20)
  ↓
用户类型生成 (top 10)
  ↓
用户选择
  ↓
对话生成
  ↓
报告输出
```

### 安全机制

```
场景库数据 (JSON)
  ↓
Fernet 对称加密
  ↓
加密文件 (.json.enc)
  ↓
Git 可安全提交
  ↓
运行时解密 (需要密钥)
  ↓
密钥文件 (~/.virtual_user/.key)
  ↓
不提交到 Git
```

## 📖 使用方式

### 命令行

```bash
# 基本用法
python src/main.py "用户的问题"

# 自动模式
python src/main.py "用户的问题" --auto

# 使用快速启动脚本
./run.sh "用户的问题"
```

### Python 代码

```python
from src import VirtualUserSkill

skill = VirtualUserSkill()
result = skill.run("用户的问题", auto_select=True)

print(f"生成用户类型：{len(result['user_types'])} 个")
print(f"对话轮数：{result['conversation_summary']['conversation_count']} 轮")
```

### 工作流/Coze 集成

```python
def virtual_user_interview(question: str):
    skill = VirtualUserSkill()
    result = skill.run(question, auto_select=False)
    
    # 返回用户类型列表供用户选择
    return {
        'user_types': result['user_types'],
        'next_step': 'wait_for_user_selection'
    }
```

## 🔒 数据保护

### 保护级别

- ✅ 防止普通用户直接读取场景库
- ✅ 防止意外泄露用户隐私
- ✅ 加密数据可安全提交到 GitHub
- ⚠️ 不防专业技术人员破解（如需更高级别，使用 API 服务）

### 密钥管理

```
首次运行 → 自动生成密钥 → 保存到 ~/.virtual_user/.key
  ↓
密钥权限 600（仅所有者可读写）
  ↓
加密场景库 → 加密文件提交到 Git
  ↓
运行时自动解密 → 不暴露原始数据
```

## 📊 场景库数据

### 数据来源

- 语雀文档：https://aliyuque.antfin.com/rlnbbu/ic0cde/ic7006fdgz33cyqh
- 数据量：1859 条用户场景
- 字段：12 个（序号、用户背景、场景、任务、痛点等）

### 数据格式

**原始字段**：
- 序号
- 用户姓名和背景信息
- 内容范围
- 是否出境游
- 用户具体场景
- 任务
- 期待效果
- 当前方案
- 爽点
- 痛点
- 改进方向
- 底层需求

**向量化字段**（用于相似度搜索）：
- 用户具体场景
- 任务
- 痛点
- 底层需求
- 用户背景信息
